# Napcat Adapter for Maibot-java

适用于 Maibot-JE 的 Napcat 协议适配mod

## 功能（开发中）

- 支持 Napcat 协议的聊天机器人接入 Maibot-JE

## 安装

1. 构建Jar包：
   ```bash
   ./gradlew jar
   ```
   这将在`build/libs/`目录下生成 `napcat_adapter-x.x.x.jar` 文件。
2. 将生成的 `Napcat-Adapter-x.x.x.jar` 文件放入 Maibot-JE 的 `mods` 目录下。q
